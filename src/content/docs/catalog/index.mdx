Over a thousand building blocks for the tools your community needs — the same ones
the And Other Stuff crew ships freedom tech with, each **license-checked at a
commit** and screened so it can't quietly betray the people you build for — plus
**800+ public datasets** indexed from
[awesome-public-datasets](https://github.com/awesomedata/awesome-public-datasets).
Use the **Kind** filter to switch between tools and datasets. Filter live below,
or hand a shortlist straight to the [Build Studio](/build/).
