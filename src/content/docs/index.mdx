## A harness gives you hands. This gives you judgment.

Goose, Claude Code, Aider — these are **harnesses**. They connect an AI model to
your files and tools so it can execute. They don't know *what* you should build,
*which* dependencies are safe, or *what license quietly changed last week*. You
bring the intent, the architecture, the policy, and the verification. The harness
brings the hands.

**We Can Just Build Things is the layer above the harness:** a verified dependency
catalog, an exclusion policy enforced by tooling, a guided decision flow, and
reusable agent skills — so when an agent finally runs, it runs inside a frame of
decisions that have already been made, verified, and documented.

## Who this is for

This is **not** built for the founder who already controls the agent network and
sets the norms. It's built for the organizer, the nonprofit, the collective — the
person who has to *trust* the system without being able to see inside it, and who
needs to make architectural decisions about what they depend on with their eyes
open.
